# EasyPeasyLogo – AI Brand Identity Designer

## Overview

EasyPeasyLogo is a fully functional AI‑driven brand identity platform.  
Users converse with an AI designer, define their brand direction, and receive professional logos, banners, thumbnails, and other visuals.  
The entire system is pre‑configured and ready to run – you just need to connect your own API keys and hosting accounts, or choose the white‑glove setup where I handle everything for you.

## Google OAuth Setup – Quick Overview

The platform uses "Sign in with Google". This requires:
1. Creating OAuth credentials in Google Cloud Console.
2. Adding the Supabase callback URL to the credentials.
3. Pasting the Client ID and Secret into Supabase → Authentication → Providers → Google.

**Full step‑by‑step instructions are included in the `handover/` setup guide.**
